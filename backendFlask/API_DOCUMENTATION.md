# API Documentation for CapitalTwo Project

## Endpoints you would need `Authorization: Bearer {`Token`}` for:

### Budget
- `POST` budget
- `DELETE` budget
- `PUT` budget

### Transaction
- `POST` transaction
- `DELETE` transaction
- `PUT` transaction

### Subscription
- `POST` subscription
- `DELETE` subscription
- `PUT` subscription

### User
- `POST` logout (no params)
- `DELETE` user
